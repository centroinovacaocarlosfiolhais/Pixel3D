🧱 O que são Voxels?

Voxel = Volume + Pixel — um ponto no espaço tridimensional.

💡 Pensa assim:
  • Um pixel é um ponto numa imagem 2D (plano)
  • Um voxel é um cubo num espaço 3D
  • Juntos, formam qualquer forma 3D que imagines!

📐 Coordenadas 3D — O Mapa do Espaço

Para colocar um bloco no espaço, precisamos de três coordenadas:

↔️

X — Largura

Da esquerda para a direita
0 a 15

↕️

Y — Altura

Do chão para o céu
0 a 15

↙️↗️

Z — Profundidade

Da frente para trás
0 a 15

🎮 Dica Gamer:

É como um mapa de jogo! X e Z são o mapa 2D, Y é o andar em que estás. No Pixel3D, usas o seletor de camada (Y) para escolher o "andar" onde constróis.

// Um bloco na posição (5, 3, 7) significa:
X = 5   // 5 casas da margem esquerda
Y = 3   // 3º andar (camada)
Z = 7   // 7 casas do fundo
⚠️ O Y = 0 é sempre o chão! Para construir em cima, aumenta o Y com as setas do Painel.

🎨 A Paleta PICO-8 — 16 Cores, Infinitas Possibilidades

O Pixel3D usa a paleta do PICO-8 — uma consola retro virtual com apenas 16 cores. Esta limitação é intencional!

💡 Porque limitar as cores?
  • Obriga a fazer escolhas criativas
  • Cria uma identidade visual consistente
  • É o mesmo princípio do pixel art retro
  • Todos os modelos ficam no mesmo universo visual
🎮 Curiosidade:

O PICO-8 é uma "consola fantasma" criada em 2015. Tem 128×128 pixels, 16 cores e é usada para criar jogos retro. É muito popular entre game developers iniciantes!

🎯 Desafios — Escolhe o Teu Nível

🟢 Iniciante

Personaliza o Template!

  • Escolhe um template (Humanóide, Animal, etc.)
  • Muda a cor de pelo menos 5 blocos
  • Adiciona 3 novos blocos

⏱️ 10-15 min

🟡 Intermédio

Cria um Animal Reconhecível

  • Apaga o template e começa do zero (ou de um template)
  • Cria um animal que qualquer colega reconheça
  • Usa pelo menos 3 camadas (Y) diferentes

⏱️ 20-30 min

🔴 Avançado

Cria uma Personagem com Detalhes

  • Personagem com acessórios (chapéu, arma, mochila…)
  • Usa pelo menos 5 cores diferentes
  • Exporta o GIF e partilha

⏱️ 30-40 min

🟣 Mestre

Cria algo que não existe na Terra

  • Monstro, nave, robot, criatura alienígena…
  • Sem limites — liberdade total!
  • Apresenta à turma

⏱️ 40+ min

💡 Dicas de Mestre

✅ Boas Práticas

  • Começa pelo chão (Y=0) e vai subindo
  • Usa o plano verde para orientação — é a camada ativa
  • Roda a câmera com frequência para ver todos os ângulos
  • Varia as cores por zonas para criar contraste
  • Simplifica! Menos blocos = mais impacto visual
  • Experimenta sem medo — apagar é fácil!

❌ Armadilhas Comuns

  • Camada errada — verifica sempre o Y antes de clicar
  • Câmera perdida — clica em "🎥 Reset" no painel
  • GIF demora — é normal! São 24 frames a renderizar
  • Demasiados blocos — pode ficar confuso, simplifica
  • Desistir cedo — os primeiros 5 minutos são os mais difíceis!

📄 Momento Surpresa — O Ficheiro TXT!

O Pixel3D pode exportar o teu modelo como texto simples. Cada camada aparece como uma matriz 2D de letras!

════════════════════════════════════
  PIXEL3D — O Teu Modelo Voxel
════════════════════════════════════
  Blocos  : 42
  Grid    : 16×16×16

  Legenda: · = vazio, letra = cor

━━━ Y=0 ━━━
  · · · · · Z Z Z · · · · · · · ·
  · · · · Z Z Z Z Z · · · · · ·
  · · · · · Z Z Z · · · · · · ·

━━━ Y=1 ━━━
  · · · · · · R R · · · · · · ·
  · · · · · · R R · · · · · · ·
🧠 O que isto ensina?
  • Arrays 2D — uma grelha de valores
  • Indexação — posição [x][z]
  • Representação de dados — o computador guarda assim
🎯 Como usar na sessão:

Exporta o TXT ao vivo e projeta. Mostra que o teu modelo são apenas números e letras. É o mesmo princípio de qualquer videojogo!
